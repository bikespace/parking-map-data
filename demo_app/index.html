About the Data

Please see Github Repo. Uses the OpenStreetMap tagging scheme.

Important values include:

  • bicycle_parking
    : type of bicycle parking, e.g. 
    bollard
    for ring and post, 
    rack
    for rack, etc.
  • capacity
    : number of bikes that can be locked to the parking feature
  • cargo_bike
    : whether there is enough space (and the right kind of locking object) to lock up a cargo bike

Legend

  • Blue dots are from OpenStreetmap
  • Black dots are from City of Toronto Data
  • Smaller dots are Stands/Bollards (since these are more frequent, and individually tend to be lower-capacity)
